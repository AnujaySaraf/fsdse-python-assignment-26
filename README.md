# Write a function to remove a key from a dictionary.

Define a function that takes two parameters: a dictionary and a key.
Then remove the key from the dictionary if it is present. Return the dictionary.

_Example1:_

**Input:**

`{1:10, 2:20, 3:30}, 2`


**Output:**

`{1:10, 3:30}`


_Example2:_

**Input:**

`{1:10, 2:20, 3:30}, 4`


**Output:**

`{1:10, 2:20, 3:30}`
